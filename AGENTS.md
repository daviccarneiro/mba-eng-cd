# AGENTS.md

## Context
- This is an Obsidian vault for the MBA in Engenharia e Ciência de Dados.
- Start date: 2026-04-25.
- Institution: UNIESP Centro Universitário.

## Work In This Vault
- Keep notes, images, and Excalidraw files in the vault; do not introduce app-style build or test workflows.
- Preserve existing notes and links. Avoid renaming or moving files unless the vault structure explicitly needs it.
- Use relative Obsidian links and transclusions as-is; check link targets before changing paths.

## Folder Rules
- Keep empty directories tracked with `.gitkeep`.
- Create new folders before adding notes into them.
- Use these top-level buckets for new content:
  - `00-entrada/` for raw captures and temporary notes.
  - `01-aulas/` for class notes and meeting notes.
  - `02-disciplinas/` for topic and subject notes.
  - `03-projetos/` for assignments, cases, and deliverables.
  - `04-referencias/` for summaries, links, and source material.
  - `90-anexos/` for images, diagrams, and other attachments.

## Notes For Future Changes
- Keep dated notes grouped consistently with the existing vault convention.
- Do not delete `.gitkeep` files when a directory is otherwise empty.
