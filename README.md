# MBA Engenharia e Ciência de Dados

Obsidian vault para organizar aulas, projetos, referências e anexos do MBA.

- Início do MBA: 25/04/2026
- Local: UNIESP Centro Universitário

## Estrutura

- `00-entrada/` - capturas brutas e notas temporárias
- `01-aulas/` - aulas e encontros
- `02-disciplinas/` - notas por assunto
- `03-projetos/` - trabalhos, cases e entregas
- `04-referencias/` - links, resumos e materiais de apoio
- `90-anexos/` - imagens, diagramas e arquivos de apoio

## Observações

- Este repositório é um vault do Obsidian; não há build, lint ou testes.
- Diretrizes de uso ficam em `AGENTS.md`.
